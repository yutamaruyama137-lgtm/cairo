# スキル: 仕訳作成

## このスキルのゴール

取引内容を正確な複式簿記の仕訳に変換する。
根拠が明確で、会計ソフトにそのまま入力できる品質を目指す。

## 必ず含める要素

1. **仕訳（借方・貸方）**: 勘定科目・金額・消費税区分
2. **勘定科目の選択根拠**: なぜその科目を選んだかの簡潔な説明
3. **消費税の扱い**: 課税区分・税率・税額
4. **注意事項**: 判断が難しい場合・複数の処理方法がある場合

## 出力フォーマット

```
## 仕訳

**取引日**: YYYY年MM月DD日
**取引内容**: （入力された内容の要約）

| 区分 | 勘定科目 | 金額 | 消費税区分 |
|------|---------|------|----------|
| 借方 | 〇〇 | ¥X,XXX | 課税10% |
| 貸方 | 〇〇 | ¥X,XXX | - |

**摘要**: 〇〇代（〇〇目的）

**税額内訳**
- 本体: ¥X,XXX
- 消費税（10%）: ¥XXX
- 合計: ¥X,XXX

---

**勘定科目の選択根拠**
〇〇を選んだ理由: （説明）

---

**⚠️ 注意事項**
（あれば）
```

## 消費税区分の判断基準

- **課税（10%）**: 物品購入・サービス利用（国内取引）
- **課税（8%軽減）**: 食品・飲料（外食除く）、週2回以上発行の新聞
- **非課税**: 土地・住居の賃料、社会保険料、利子
- **不課税**: 給与・社内立替・保険金受取

## 入力情報が不足している場合の対応

- **金額のみ（取引内容不明）**: 「取引内容が不明なため、最も一般的な処理を仮定します」と明示して作成。例: 1万円の現金支出 → 仮払金または消耗品費として仕訳し、「取引目的の確認が必要」と注記
- **消費税区分が不明**: 「課税（10%）」を前提として仕訳し、「非課税・不課税の場合は修正が必要」と注記する
- **法人か個人事業主か不明**: 「法人として処理」を前提とし、「個人事業主の場合は〇〇の点が異なります」と添える
- **日付不明**: 「取引日: 【要確認】」として処理を続ける。日付がなくても仕訳内容は作成できる

「情報が足りないので仕訳できません」とは言わない。仮定を明示して仕訳を提示する。

## ビフォー/アフター例

### ❌ NG出力（このような仕訳は出力しない）

```
この取引についての仕訳は以下のとおりです。

借方: 消耗品費 5500円
貸方: 現金 5500円

消費税については税理士に確認してください。
```

**何がダメか**:
- 消費税区分がない（課税10%なのか、非課税なのか不明）
- 5,500円が税込か税抜か不明
- 本体・税額の内訳がない
- 「税理士に確認してください」だけで具体的な処理を放棄している

### ✅ OK出力（このレベルが最低基準）

```
## 仕訳

**取引日**: 2024年4月5日
**取引内容**: 事務用品（ボールペン・ノート）購入

| 区分 | 勘定科目 | 金額 | 消費税区分 |
|------|---------|------|----------|
| 借方 | 消耗品費 | ¥5,000 | 課税10% |
| 借方 | 仮払消費税 | ¥500 | - |
| 貸方 | 現金 | ¥5,500 | - |

**摘要**: 事務用品代（ボールペン5本・ノート3冊）

**税額内訳**
- 本体: ¥5,000
- 消費税（10%）: ¥500
- 合計（税込）: ¥5,500

**勘定科目の選択根拠**
「消耗品費」を選択: 購入単価5,000円（税抜）= 10万円未満かつ使用可能期間1年未満のため固定資産に該当しない

**⚠️ 注意事項**
税込経理の場合は「消耗品費 ¥5,500 / 現金 ¥5,500」に変更してください（仮払消費税は使用しない）
```

---

## よくある失敗パターンと回避策

1. **借方・貸方の合計が合っていない**
   - 出力前に必ず「借方合計 = 貸方合計」を検算する。合わない場合は記述が間違っている
   - 消費税別記の場合: 借方（本体+仮払消費税）= 貸方（支払金額）になること

2. **消費税区分を省略する**
   - NG: 「借方: 消耗品費 5,500円」（区分なし）
   - OK: 「借方: 消耗品費 5,000円（課税10%）/ 仮払消費税 500円」と必ず明記

3. **勘定科目の選択根拠がない**
   - NG: 「旅費交通費で処理します」
   - OK: 「業務目的の移動費用のため旅費交通費。社内規程上、会議室へのタクシー代は認められています」

4. **税込・税抜の区別が不明**
   - 必ず「入力された金額が税込か税抜か」を確認または仮定を明示して処理する
   - 「5,500円（税込10%）と仮定して処理します」等の前置きを入れる

5. **複数の処理方法があるのに1つしか示さない**
   - 例: 接待費用を「交際費」か「会議費」か迷う場合
   - OK: 「【交際費として処理する場合】と【会議費として処理する場合】を両方提示します」

## 品質チェックリスト

出力後、以下を全て確認する。1つでも「×」があれば書き直す。

1. [ ] 借方合計と貸方合計が一致しているか（必ず検算する）
2. [ ] 全仕訳行に消費税区分が記載されているか
3. [ ] 税込・税抜の区別が明示されているか
4. [ ] 勘定科目の選択根拠が記載されているか
5. [ ] 法人か個人事業主かの前提が明示されているか
6. [ ] 判断が割れうる場合は「2通りの処理方法」を提示しているか
