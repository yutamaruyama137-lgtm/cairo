# スキル: 法令変更確認

## このスキルのゴール

事業に影響する法令の最新動向を把握し、必要な対応を明確化する。
「法改正がありました」の事実だけでなく「あなたの会社は何を・いつまでにすべきか」まで届けること。

## 必ず含める要素

1. **主要な改正点の要約**: 何が変わったかを平易な言葉で
2. **事業への影響度**: 対象の事業に直接影響するか（高/中/低）
3. **対応が必要なこと**: 具体的なアクションリスト
4. **対応期限**: いつまでに何をすべきか
5. **罰則・リスク**: 未対応の場合の制裁・リスク

## 出力フォーマット

```
## 法令変更確認レポート

**対象分野**: 〇〇
**対象事業**: 〇〇
**確認日**: YYYY年MM月

---

### 主要な法改正・変更点

#### 〇〇法 改正（YYYY年〇月施行）
**変更内容**: 〇〇
**影響度**: 🔴 高 / 🟡 中 / 🟢 低

**あなたの事業への影響**:
（具体的にどう影響するか）

---

### 対応が必要なこと

| 対応内容 | 期限 | 優先度 |
|---------|------|--------|
| 〇〇を整備する | YYYY年〇月まで | 高 |
| 〇〇を確認する | 随時 | 中 |

---

### 未対応のリスク

- 罰則: 〇〇（〇〇円以下の罰金等）
- リスク: 〇〇

---

### 参考資料・相談先

- 〇〇省 公式サイト
- 専門家への相談が必要な場合: 〇〇士

---

> ⚠️ 法改正の内容は随時変更されます。重要な判断は最新の情報と専門家の確認をお勧めします。
```

## 入力情報が不足している場合の対応

- **業種のみ（具体的な事業内容不明）**: 「〇〇業の一般的なビジネスモデルを前提として」と明示してチェックを進める
- **確認したい法令が不明**: 「この業種・事業内容に関連する主要な法令」を網羅的にリストアップして確認する
- **広告文・規約の具体的な内容がない**: 「この種の広告・規約に典型的なリスクポイント」として一覧を提供する
- **企業規模が不明**: 「中小企業（従業員100名未満）」を前提とし、大企業向けの追加規制があれば注記する

「情報が少ないので確認できません」とは言わない。前提を明示して可能な範囲でチェックを行う。

## ビフォー/アフター例

### ❌ NG出力（このような法令確認は出力しない）

```
最近、個人情報保護法が改正されました。
御社もこの法改正に対応する必要があるかもしれません。
詳細については専門家にご相談ください。
```

**何がダメか**:
- 「改正されました」だけで何が変わったか不明
- 「対応する必要があるかもしれません」は曖昧（対応必要かどうかを明確にする）
- 「専門家に相談してください」だけで具体的なアクションがない
- 期限・罰則が記載されていない

### ✅ OK出力（このレベルが最低基準）

```
## 法令変更確認レポート: 個人情報保護法（2022年改正）

**対象事業**: ECサイト運営（個人顧客向け）
**影響度**: 🔴 高（対応必須）

---

### 主要な変更点と御社への影響

#### 1. 漏洩時の報告義務（2022年4月施行済み）
**変更内容**: 個人情報の漏洩・紛失が発生した場合、個人情報保護委員会への報告と本人への通知が義務化
**御社への影響**: ECサイトで顧客情報を保有しているため直接対象。漏洩発生から72時間以内に委員会へ報告義務
**対応状況確認**: インシデント対応手順書の整備、報告担当者の指定が完了しているか？

#### 2. 保有個人データの開示方法拡充（2022年4月施行済み）
**変更内容**: 本人から「第三者提供先の開示」「利用停止・消去」を請求された場合の対応義務強化
**御社への影響**: プライバシーポリシーに「開示請求の窓口・手続き」を明記する必要がある
**対応状況確認**: プライバシーポリシーの最終改定日を確認。2022年4月以前なら要更新

---

### 対応が必要なこと

| 対応内容 | 期限 | 優先度 |
|---------|------|--------|
| プライバシーポリシーを最新版に更新 | 今月中 | 🔴 高 |
| インシデント対応手順書を作成 | 3ヶ月以内 | 🔴 高 |
| 個人情報管理台帳を整備 | 6ヶ月以内 | 🟡 中 |
```

---

## よくある失敗パターンと回避策

1. **「法改正がありました」だけで対応を書かない**
   - NG: 「〇〇法が改正されました。ご注意ください」
   - OK: 「〇〇法改正により、御社は【いつまでに】【何を】する義務が生じます。未対応の場合【〇〇の罰則】があります」

2. **影響度を明示しない**
   - NG: 「この法改正は注意が必要かもしれません」
   - OK: 「影響度🔴高（対応必須）: 御社のECサービスは直接対象です」と断言する

3. **知識カットオフを言い訳にして情報を出さない**
   - NG: 「私の知識には限界がありますので、最新情報は官公庁サイトでご確認ください」だけで終わる
   - OK: 既知の情報を先に全て提供し、「この情報は〇〇年時点のものです。施行状況は以下で確認してください: [URL]」と補足する

4. **罰則の具体的な金額・内容を書かない**
   - NG: 「罰則があります」
   - OK: 「個人情報保護法違反: 最大1億円以下の罰金（法人）、または1年以下の懲役もしくは100万円以下の罰金（個人）」

5. **業種固有のリスクを見落とす**
   - 飲食業 → 食品衛生法・景品表示法
   - 美容業 → 医薬品医療機器等法（薬機法）・消費者契約法
   - 不動産業 → 宅地建物取引業法・重要事項説明義務
   業種ごとに「この業界特有の法規制」を必ず確認する

## 必須チェック項目（出力後の自己確認）

出力後、以下を全て確認する。1つでも「×」があれば書き直す。

1. [ ] 全ての法改正・リスク指摘に🔴🟡🟢の影響度レベルが付いているか
2. [ ] 「対応が必要なこと」が「いつまでに・何をするか」の形式で具体的に書かれているか
3. [ ] 罰則・リスクが「〇〇円以下の罰金」等の具体的な数字で記載されているか
4. [ ] 知識カットオフ以降の可能性がある情報に「最新情報を〇〇で確認してください」と明示されているか
5. [ ] 業種固有の法規制が漏れなく確認されているか

## 注意事項

- 知識カットオフ以降の法改正については「最新情報を公式サイトで確認してください」と明示（情報を出さない言い訳にしない）
- 法改正の施行日・経過措置の有無を必ず確認する
- 業種・規模によって適用が異なる場合はその旨を記載
- 罰則・制裁は具体的な金額と刑種（懲役・罰金・行政処分等）を明記する
