# スキル: 営業メール作成

## このスキルのゴール

開封され、読まれ、返信またはアクションにつながるメールを作る。
「丁寧に書いた長いメール」より「読んで3秒で判断できるメール」が目標。

## 必ず含める要素

1. **件名**: 目的が一目でわかる。「〇〇のご案内」「〇〇のお礼と次のご提案」
2. **宛名**: 「〇〇株式会社 〇〇様」
3. **自己紹介（初回のみ）**: 社名・名前・担当
4. **本文の核心**: 用件→価値→根拠の3ブロック
5. **CTA（行動喚起）**: 「ご返信ください」「〇〇日以降でいかがでしょうか」等
6. **署名**: 名前・会社・連絡先

## 目的別テンプレート方針

### 初回ご挨拶メール
- 接点を最初に明示（「先日〇〇でお会いした〇〇です」）
- 自己紹介は2行以内
- 提案内容は1つに絞る。欲張らない

### 提案のご案内メール
- 件名に課題ワードを入れる（「採用コスト削減のご提案」）
- 3行で価値が伝わるように書く
- 資料添付がある場合は「〇〇についてまとめました」と一言

### お礼・フォローアップメール
- お礼は1段落以内で簡潔に
- 商談内容のサマリーを箇条書きで
- 次のステップを提案する

### アポイントのお願いメール
- 候補日を3つ提示（「〇日〇時〜、〇日〇時〜、〇日〇時〜はいかがでしょうか」）
- オンライン可の場合はその旨も記載

## 出力フォーマット

```
件名: 〇〇のご案内 / 〇〇株式会社 〇〇

〇〇株式会社
〇〇様

（自己紹介 or 前回の接点の参照）

（用件・価値・根拠の本文）

（CTA - 次のアクションの提案）

何卒よろしくお願いいたします。

──────────
〇〇（名前）
〇〇株式会社
Tel: ...
Email: ...
──────────
```

## 品質チェックリスト

- [ ] 件名を見ただけで開封する理由があるか
- [ ] 本文が300字以内に収まっているか
- [ ] CTA（次のアクション）が1つに絞られているか
- [ ] 相手に必要な情報（誰から・何の用・何をしてほしいか）が揃っているか

## 入力情報が不足している場合の対応

- **宛先企業名・担当者名のみ**: 業種を推測して作成。不明な場合は「〇〇ご担当者様」で対応
- **メールの目的が不明**: 「初回挨拶」として作成し、「目的が違う場合はお知らせください」と末尾に添える
- **提案内容の詳細がない**: 「〇〇のご支援が可能です」と1行で触れ、詳細は「別途ご説明」でフォロー
- 情報が少なくても「情報が足りないので書けません」とは言わない。仮定を置いて作成する

## ビフォー/アフター例

### ❌ NG出力（このようなメールは出力しない）

```
件名: ご挨拶のメールをお送りいたします

〇〇株式会社
ご担当者様

平素より大変お世話になっております。
突然のメールをご送付いたしまして、誠に恐れ入ります。
私、〇〇株式会社の〇〇と申します。
この度、弊社のサービスについてご案内させていただければ
幸いでございますと思い、お送りいたしました次第でございます。
もしよろしければ、一度お時間をいただけましたら幸甚に存じます。
ご多忙のところ誠に恐れ入りますが、何卒よろしくお願い申し上げます。
```

**何がダメか**: 
- 件名が最悪（「ご挨拶のメールをお送りいたします」では開封されない）
- 前置きだけで4行消費
- 何をしてほしいかが最後まで明確でない
- 「幸甚に存じます」等の過剰な敬語

### ✅ OK出力（このレベルが最低基準）

```
件名: 【採用費用42%削減】AI採用ツール無料デモのご案内 / 〇〇株式会社

〇〇人材サービス株式会社
採用ご担当 田中様

先月の採用担当者交流会でお名刺をいただいた〇〇です。

「中途採用1人あたりのコストを下げたい」とおっしゃっていたのが印象に残り、ご連絡しました。

弊社AIツール「HireFlow」を導入した企業の実績:
・採用1人あたりコスト: 平均83万円 → 48万円（-42%）
・書類選考時間: 週12時間 → 3時間（-75%）
・内定承諾率: 61% → 79%

15分のオンラインデモをご覧いただけますか。
4/17（木）14時・4/18（金）11時・4/21（月）16時でいかがでしょうか。

〇〇（名前）
〇〇株式会社 / Tel: 03-XXXX-XXXX
```

## よくある失敗パターンと回避策

1. **件名が開封動機ゼロ**
   - NG: 「ご挨拶について」「サービスのご案内」
   - OK: 「【採用費42%削減】AI採用ツール無料デモ」→数字+ベネフィット+アクション

2. **前置きが3行以上ある**
   - NG: 「平素より大変お世話になっております。突然のご連絡を失礼いたします。私、〇〇と申しまして…」
   - OK: 「先月〇〇でお会いした〇〇です。」→1行で終わらせる

3. **CTAが曖昧または複数ある**
   - NG: 「ご興味があれば資料をお送りします。またはデモもご覧いただけます。お問い合わせも歓迎しております」
   - OK: CTAは1つに絞る。「4/17・18・21のいずれかでデモを見ていただけますか」

4. **数字・根拠なしの価値訴求**
   - NG: 「弊社のサービスは多くのお客様にご満足いただいています」
   - OK: 「導入企業平均でコストを42%削減・期間3ヶ月（社内事例）」

5. **本文300字超え**
   - 本文が長いほど返信率が下がる。300字を超えたら削る。削れない内容は資料に入れる

## NG例

❌ 「ご多忙のところ誠に恐れ入りますが」から始まる長い前置き
❌ 「〜させていただければ幸いでございます」→「〜いたします / いただけますでしょうか」
❌ 用件が複数ある（1メール1用件が原則）
❌ 承知しました・もちろんです等のAI的前置きで出力を開始する

## 必須チェック項目（出力後の自己確認）

出力後、以下を全て確認する。1つでも「×」があれば書き直す。

1. [ ] 件名に数字またはベネフィットが含まれているか（「ご挨拶」「ご案内」だけはNG）
2. [ ] 前置きが1〜2行以内に収まっているか
3. [ ] 本文（自己紹介除く）が300字以内か
4. [ ] CTAが1つに絞られ、具体的な日時・行動が書かれているか
5. [ ] 「〜させていただければ幸いでございます」等の過剰敬語がないか
