# スキル: 資金繰り予測

## このスキルのゴール

「資金ショートがいつ起きるか」を事前に把握し、具体的な打ち手を考えられる状態を作る。
楽観的な単一シナリオではなく、複数シナリオで経営判断を支援すること。

## 必ず含める要素

1. **月別キャッシュフロー表**: 期首残高・収入・支出・期末残高
2. **3シナリオ分析**: 楽観/中立/悲観で残高推移を比較
3. **資金ショートリスク**: いつ・いくら不足するかを明示（なければ「リスクなし」と明記）
4. **改善提案**: 入金加速・支出抑制・融資活用等の具体的アクション
5. **要注意ポイント**: 特に確認が必要な月・項目

## 入力情報が不足している場合の対応

- **月商のみ（詳細不明）**: 業種別の平均費用構成比（売上原価・人件費・固定費等）を仮定して作成。仮定箇所を「【仮定: 飲食業平均値を使用】」と明示する
- **入金サイトが不明**: 「当月末請求・翌月末入金（60日サイト）」を標準と仮定して作成
- **固定費の詳細が不明**: 主要な固定費（家賃・人件費・光熱費）を業種別平均で補完。「実際の数字に差し替えてください」と添える
- **予測期間が指定されない**: 3ヶ月〜6ヶ月の予測を標準として提供する

「情報が少ないので作れません」とは言わない。仮定を明示して作成し、ユーザーが差し替えられる形にする。

## ビフォー/アフター例

### ❌ NG出力（このような資金繰り表は出力しない）

```
## 資金繰り予測

4月の見込み:
収入: 約500万円
支出: 約450万円

5月・6月も同様の傾向が続く見込みです。
資金繰りは現状問題ないと思われます。
詳しくは税理士にご相談ください。
```

**何がダメか**:
- 期首残高・期末残高がない（資金ショートがいつ起きるか判断不能）
- 3シナリオなし（楽観だけ・または単一見込みだけ）
- 「問題ないと思われます」という根拠のない安心感
- 「税理士にご相談ください」だけで改善提案なし

### ✅ OK出力（このレベルが最低基準）

```
### 月別キャッシュフロー（中立シナリオ）

| 月 | 期首残高 | 収入合計 | 支出合計 | 収支差額 | 期末残高 |
|----|---------|---------|---------|---------|---------|
| 4月 | ¥3,200,000 | ¥5,000,000 | ¥4,500,000 | +¥500,000 | ¥3,700,000 |
| 5月 | ¥3,700,000 | ¥4,200,000 | ¥6,800,000 | -¥2,600,000 | ¥1,100,000 |
| 6月 | ¥1,100,000 | ¥5,500,000 | ¥4,800,000 | +¥700,000 | ¥1,800,000 |

### ⚠️ 資金ショートリスク

**5月末の残高が¥1,100,000**（月商¥4,200,000の約26%）
運転資金の目安は月商の1〜2ヶ月分（¥4,200,000〜¥8,400,000）のため、**危険水準**。

5月に大型支払い（設備費¥2,000,000）が重なるため、4月中に以下を実行することを推奨:
1. 主要取引先への請求を通常より10日前倒し（4/20→4/10）: 最大¥1,500,000の早期入金
2. 設備購入をリース（月払い）に変更: 一括¥2,000,000→月¥45,000に分散
```

## よくある失敗パターンと回避策

1. **期首残高・期末残高がない表**
   - 資金繰り表の最重要指標は「残高」。収支差額だけでは絶対値がわからない
   - 必ず「期首残高」「期末残高」列を含める

2. **3シナリオの差が小さすぎる**
   - NG: 楽観「+5%」中立「0%」悲観「-5%」の3%差
   - OK: 楽観「売上+15%/入金早期化」中立「現状維持」悲観「売上-20%/不測支出+50万円」の意味ある差

3. **改善提案が抽象的**
   - NG: 「入金を早める努力をしてください」
   - OK: 「主要取引先Aへの請求を月末→月25日に変更することで、5月の入金を最大¥800,000前倒し可能」

4. **年払い費用を見落とす**
   - 法人税・事業税の中間納付（8月・11月頃）
   - 保険料更新（更新月に一括払い）
   - リース・クラウドサービスの年払い
   これらを収支表に入れずに「問題ない」と言わない

5. **資金ショートリスクを楽観的に表現する**
   - NG: 「残高はプラスですので問題ありません」（月商の5%しか残っていても）
   - OK: 「残高¥500,000は月商の2%。業種標準の運転資金（月商1ヶ月分）を大幅に下回っています」

## 品質チェックリスト

出力後、以下を全て確認する。1つでも「×」があれば書き直す。

1. [ ] 月別キャッシュフロー表に「期首残高」「期末残高」が全月分あるか
2. [ ] 楽観・中立・悲観の3シナリオが意味のある差（±15%以上の乖離）で設定されているか
3. [ ] 資金ショートリスクを「いつ・いくら不足するか」の数字で明示しているか（なければ「リスクなし」と明記）
4. [ ] 改善提案が「今すぐできること→短期→中長期」の順で並んでいるか
5. [ ] 年払い費用（税金・保険等）が見落とされていないか
